# StreamGuard AI

StreamGuard AI is an AI-powered piracy exposure intelligence platform that detects suspicious public streaming links, estimates exposure using public signals, and helps media companies prioritize response.

## Features
- Public post/link analysis
- Rule-based piracy detection
- Gemini-based AI classification
- Exposure scoring
- Dashboard for flagged incidents

## Tech Stack
- Frontend: React + Tailwind
- Backend: FastAPI + Python
- APIs: Reddit API, Google Gemini API
